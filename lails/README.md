# Lails

## 概要

Railsの学習のため、Railsと似たような動作をしてくれるRubyのプログラムを作成する

## 要件・制約

* Railsチュートリアルで作ったWEBアプリケーションが動くようにする
* Railsのソースコードは読まない
* テスト・アセットパイプラインは再現の対象に含めない

## 開発対象

* サーバ待ち受け
* ルーティング
* コントローラの基底クラス `ActionController::Base` 
* モデルの基底クラス `ActiveRecord::Base` 
* 各種のヘルパー
* DB操作
* セッション管理
* Cookieの管理
* erbの描画
+ 静的リソースの提供
 
## ファイル構成

* `server.rb` 起動に使うファイル

## コーディング規約

* 勝手に実装したメソッドは `_` から始めることにする
